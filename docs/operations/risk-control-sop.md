# 风控与纠纷处理 SOP（V1）

## 1. 风险类型
- 设备来源风险（序列号异常、来源不明）
- 交易风险（虚假报价、虚假挂单、恶意压价）
- 履约风险（超时未服务、报告造假、乱收费）
- 账号风险（刷单、批量账号、恶意评价）

## 2. 风险分级
- P1：高风险，立即冻结相关交易
- P2：中风险，限制曝光并人工复核
- P3：低风险，记录观察并持续跟踪

## 3. 处置时效
- P1：30 分钟内响应，4 小时内出结论
- P2：4 小时内响应，24 小时内出结论
- P3：24 小时内响应，72 小时内关闭

## 4. 纠纷处理流程
1. 用户提交争议单并上传证据
2. 平台收集检测记录、聊天记录、报价记录
3. 风控员判责并给出处置建议
4. 执行补偿/扣分/下架/封禁
5. 复盘并更新规则

## 5. 风控闭环
- 周度输出风险周报
- 月度更新风控规则阈值
- 高发问题同步产品优化
