# 首城试运营执行手册（V1）

## 1. 目标
- 在深圳、上海双城启动试运营，验证“检测-维保-交易”闭环。
- 8 周内形成稳定门店供给、稳定询价流量、可复盘数据体系。

## 2. 组织分工
- 城市运营：门店签约、培训、履约巡检
- 平台运营：审核、派单策略、活动运营
- 风控运营：异常订单、纠纷处理、资质抽检
- 产品技术：系统支持、问题修复、数据看板

## 3. 周度节奏
- 周一：经营复盘（上周 KPI、异常、改进项）
- 周二-周四：门店走访、流程抽检、培训
- 周五：增长专题（内容与活动）
- 周末：用户咨询高峰值班与纠纷处理

## 4. 首批门店上线流程
1. 资质提交与审核
2. 门店后台账号开通
3. 检测标准与报价规则培训
4. 试单 5 单验收
5. 正式接单

## 5. 试运营验收门槛
- 门店活跃率 >= 70%
- 工单准时完结率 >= 85%
- 检测报告合规率 >= 95%
- 有效纠纷率 <= 3%
