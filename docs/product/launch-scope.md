# 机器人平台首发范围定义（V1）

## 1. 首批聚焦品类

### 消费级
- 扫地机器人（含扫拖一体）
- 擦窗机器人
- 割草机器人（以城市别墅区/郊区场景为补充）

### 商用级
- 室内配送机器人（餐饮、酒店、园区）
- 商用清洁机器人（商场、写字楼、医院公共区域）

## 2. 首发城市
- 第一优先：深圳、上海
- 第二优先：杭州、广州、苏州（第 2 阶段扩展）

选择依据：
- 服务机器人保有量与更新速度较高
- 本地维修与门店生态较成熟
- 用户对二手交易与上门服务接受度高

## 3. 首批门店规模目标
- 0-3 个月：签约 20 家合作门店（深圳 12，上海 8）
- 3-6 个月：扩展到 50 家合作门店
- 6-12 个月：扩展到 100 家合作门店（进入第二优先城市）

门店准入标准：
- 至少 2 名专职维修技师
- 具备机器人拆机与基础电控维修能力
- 可提供标准化检测留痕（图像/视频）
- 接受平台统一报价模板与服务SLA

## 4. 品牌合作边界（首期）

### 可直接支持（优先）
- 主流消费机器人品牌（扫地、擦窗）常见机型
- 主流商用配送与清洁机型（可获取配件渠道）

### 条件支持（需评估）
- 冷门品牌或海外小众型号
- 停产机型（配件不稳定）
- 高度改装机型（非原厂结构）

### 暂不支持（首期排除）
- 来源不明或序列号无法核验设备
- 涉及关键安全风险且缺乏维修规范的机型
- 无法确认配件可得性的高故障机型

## 5. 服务边界
- 首期提供：检测、报价、维修撮合、保养、二手信息发布与询价
- 首期不提供：平台担保支付、平台自营回收翻新、跨境交易
- 后续预留：轻担保、认证整备、延保与保险

## 6. 验收指标（用于首期范围是否成功）
- 首批机型覆盖率 >= 80%（按目标城市存量机型口径）
- 首批门店 30 天活跃率 >= 70%
- 检测报告标准化合格率 >= 95%
- 二手信息发布到首次询价中位时长 <= 72 小时
